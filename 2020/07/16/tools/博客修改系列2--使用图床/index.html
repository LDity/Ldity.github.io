使用图床存储图片
发表于 2020-07-16| 更新于 2020-06-24|博客搭建系列

图床

就是存储图片在网络上，可以通过图片的外链访问图片的一种储存图片技术。

图床分类

免费的 如七牛云（10G 一个月使用时是免费的）github（想怎么用就怎么用，除了所有图片都可以被他人查看之外）等

收费的 各种阿里云 腾讯云

国外的 如这种https://cloudinary.com/ 100M内免费，我用过一段时间，速度还可以，但是太小了

使用Github

步骤一 新建仓库

1.1 点击右上角创建仓库new repository

1.2 填写仓库信息，例如我就创建了一个 Test_image的仓库。这里注意，仓库得设置为 Public 因为后面通过客户端访问算是外部访问，因此无法访问 Private ，这样的话图片传上来之后只能存储不能显示。所以要设置为 Public。如下所示
