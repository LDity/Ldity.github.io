用的顺手的软件
发表于 2020-07-16| 更新于 2020-07-07|电脑硬件系列

今天时候，想到自己以后可能会更换window电脑，换个Mac pro或者是其他的轻薄本，就自己的用的还算习惯的软件做个记录。

如果有链接，那就加上使用链接

学习系列软件

  • Sublime text3(Python)
  • VsCode(Python)
  • IDEA(Java)
  • C-Free5(C/C++)
  • HBuilderX(Html/JS)
  • Navicat Premium 12(SQL)

远程连接软件

  • Putty
  • Xshell
  • EasyConnect(进入内网)
  • ShadowsockR(VPN)

效率软件

  • Typora 笔记记录软件
  • Xmind 思维导图软件
  • Wox 本地搜索软件
  • Snipaste 本地截图软件
  • NeatDownload 多线程下载软件
  • Geek 本地彻底清除软件
  • 格式工厂 对各类格式进行转换

浏览器

  • Firefox （隐私更好）
  • Chrome
  • Micsoft Edge(PDF)
