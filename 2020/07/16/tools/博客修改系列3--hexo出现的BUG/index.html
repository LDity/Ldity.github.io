Hexo 出现的一些BUG
发表于 2020-07-16| 更新于 2020-06-30|博客搭建系列

问题：hexo d后 在GitHub中无法正常显示字体

详细问题：

我在修改了博客的Next的主题字体后，以及相应的各类东西，使用 hexo g && hexo d字体没有出现改动，但是提交上去的博客的文件却是正常显示出来了，像自己的记录的一些内容也都可以正常显示。

问题分析：

hexo s 后在本地是已经改动了自己的博客，而在hexo g改动之后提交hexo d 文档后。原本的文件可以正常显示，而字体等样式却没有得到修改。本地改动而云端没有相应的改变，一般说明云端的某些内容没有改变，所幸直接将原有的数据库删除，重新部署。

解决方式

完成自己的网站部署后，输入hexo clea && hexo g && hexo d命令，便可以了

问题：修改主题的CSS文件后，在Chrome中无法显示CSS更改后的文件

详细问题：

hexo d 上传到阿里云的服务器之后，用chrome无法显示我修改之后的CSS样式，用手机的浏览器和其他浏览器都显示正常

问题分析：

其他浏览器正常，而Chrome不正常，说明服务器是没有问题的。只能说明缓存了太多Cookie，不利于本地浏览器的显示

解决方式：

清除cookie，具体操作请看谷歌提供的cookie清除方式:Google Support
